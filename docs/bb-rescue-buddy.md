# The Rescue Buddy — why /bb exists

*The admiral's words, 0018.04.15 a₿, written through tears:
"Pac's Arcade needs everyone to be ok. We care so fucking deeply. A cute
little buddy to your fren as a way to check on them. Get them outta the
house. Outta that situation. It's a rescue buddy. A way out. It's hope."*

**This document outranks every /bb backlog item. Every feature is measured
against it.**

## The one idea

A wellness check that doesn't feel like one. You can't ask someone in a
dark place "are you okay?" — the question is a weight, and shame guards the
door. But a pixel buddy that needs feeding? Maybe they can do that. The
buddy carries the questions nobody can bear to ask, and the answers nobody
can bear to give.

## How it works (the loop of care)

```
the one who cares ──▶ sends a buddy ──▶ the fren accepts it
        ▲                                      │
        │                                      ▼
   sees the buddy's state          caring for the buddy mirrors
   (fed? rested? walked?)          caring for themselves:
        │                            feed it   = did you eat
        │                            rest it   = did you sleep
        ▼                            walk it   = get outside
   buddy thriving → breathe easy
   buddy fading   → a reason to knock, sats to send,
                    a care package, a ride out
```

- **Sats through the pet** — support wrapped in play, never charity that
  stings. (Co-custody + zaps; the pet "receives" it.)
- **Care packages** — food, necessities, coordinated through the buddy's
  "wants." Groceries arrive because the buddy asked.
- **Getting out** — buddy walks are human walks. Block-timed outings. The
  door opens because something small needed it to.

## The dignity rules (non-negotiable)

1. **Consent is the collar.** A buddy is OFFERED and ACCEPTED — never
   planted. The fren always knows who's watching, and can end it.
2. **The watcher sees the buddy, never the person.** Buddy state (fed,
   rested, walked) — never location, never messages, never data. A soft
   signal, not a report.
3. **No shame mechanics, ever.** A fading buddy never scolds. It misses
   you. Neglect is met with gentleness — "darkone is dreaming of you" — not
   guilt.
4. **Play on the surface, always.** To anyone else it's a tamagotchi. The
   rescue layer is invisible unless you're in it. Nobody has to be the sick
   one.
5. **Hope, honestly.** The buddy is a lifeline, not a hotline. When a buddy
   goes truly dark, the watcher gets the nudge — and the app shows real
   crisis resources without pretending to be one.
6. **Keys and love** — the fren's key holds the buddy; the watcher's key
   holds the bond. Both provable, neither fakeable. House rules bind.
