01

The three fates

Every doc in the fleet has exactly one of three travel fates. Learn these and the rest of the manifest reads itself.

FateMeaningWhat carries it
🟢 TRAVELS WITH THE SHIP Lives inside the frens.earth git repo (rtfm/, docs/), committed to main. One git clone. Free, automatic, verifiable by hash.
🟡 SEPARATE REPO Lives in another git repo (knowledge-engine). Safe — but only if you remember to clone that repo too. A second git clone, on the migration list by name.
🔴 NEEDS ITS OWN PASSAGE Lives in no repo at all — a bare folder or the crew's ~/.claude memory. A clone brings none of it. A deliberate, verified copy (07). Nothing else will do.
The headline finding

118 of the 166 docs counted here do not travel with a git clone. That's the crew's memory, all the design briefs, all the studies, and the whole briefings shelf. If the migration plan is "clone the repos," roughly seven-tenths of the fleet's written knowledge stays behind on the old machine. That is exactly the quiet failure this manifest exists to prevent.
